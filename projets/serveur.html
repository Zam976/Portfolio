Guide d'installation

  1. Installer Windows Server 2019.
  2. Configurer Active Directory et DNS.
  3. Créer des utilisateurs et groupes dans AD.
  4. Appliquer des stratégies via GPO pour sécuriser le réseau.
  5. Tester l'authentification sur des postes clients.
