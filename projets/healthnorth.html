Projet Health North

Présentation

Le projet Health North vise à développer un système de gestion médicale complet.

  • Gestion des patients, rendez-vous, dossiers médicaux et prescriptions.
  • Base de données relationnelle (MariaDB) pour organiser les données.
  • Accès via des applications Web, mobile et lourde.
  • Authentification sécurisée via Active Directory.
  • Gestion des versions avec GitHub.

Captures d'écran

Composantes du projet

a) Base de données

Technologie MariaDB sous WampServer.

  • Tables : Patients, RendezVous, Spécialiste, etc.
  • Procédures stockées pour gérer les patients et rendez-vous.
  • Déclencheurs pour historiser les modifications.

b) Application Web :

Technologie : PHP + MariaDB.

Fonctionnalités :

  • Gestion des patients (ajout, consultation).
  • Planification des rendez-vous.

c) Application Mobile :

Technologie : Flutter.

Consultation des dossiers patients et accès rapide aux rendez-vous.

d) Application Lourde :

Technologie : Java (Swing/JavaFX).

Fonctionnalités avancées pour gérer les prescriptions et l'import/export des données.

e) API :

Technologie : Node.js ou PHP.

Routes GET et POST pour manipuler les données.
